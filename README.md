# Cloudcap

## Prerequisites

1. [poetry](https://python-poetry.org/docs/)

## Getting started

To start, install the project dependencies:

```bash
make install
```

Enter the project's virtual environment:

```bash
make shell
```

Now you should have access to `cloudcap`:

```bash
cloudcap --help
```
